# -NFC_PCB
本项目提供了几种嘉立创（专业版）NFC_PCB，可供初学者进行学习，几种PCB均为简单的NFC，只是题材不同（有库洛米，青冥针（无NFC功能），游戏王等）。
# NFC PCB Learning Examples
本项目提供了几套基于嘉立创专业版设计的基础NFC PCB，专为初学者学习使用。所有PCB均为简单的NFC线圈与基础电路设计，仅题材与尺寸不同，可直接用于打板与测试。

---

## 📌 项目介绍
- 设计平台：嘉立创EDA（专业版）
- 适用场景：NFC入门学习、线圈参数验证、基础PCB打板练习
- 设计文件：包含原理图、PCB源文件，可直接在嘉立创EDA中打开修改
